# SHMUP

## BEFORE PLAYING
Please install and use the 'shmup.ttf' font in the ``font`` directory.

If you want to uninstall it after you can do ``fc-list "shmup"`` and remove every outputs of this command.

## HOW TO PLAY
Use ``W`` ``A`` ``S`` ``D`` to move

Use ``arrow keys`` to aim

Use ``space bar`` to shoot

## ROAD-MAP
Infinite auto-scrolling

Procedural map generation with pregenerated tiles

Custom font

Animated sprites
